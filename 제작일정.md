	•	대응 수단 추가: 경찰 신고 버튼, 체인 걸기 버튼 (UI 버튼 + 조건 분기)

	•	Secret Ending 조건 추가 (CCTV 녹화)

✅ 출력물: 2D 프로토 + 빌드업 + 대응 선택지 + 엔딩 4종
